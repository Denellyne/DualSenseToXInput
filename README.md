# Dualsense to XInput


    Translation layer for the Dualsense controller,
    Enables controller support in every game with XInput.

## Features


    Lightbar control,
    Rumble support,
    Gamecube Triggers, (Triggers with Dolphin Emulator)
    Rainbow mode on USB (Must change variable)
